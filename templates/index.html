Troubleshooting

Upload / ingest

  • Columns not detected — use the column mapper for address, title, h1.
  • Garbled URLs — re-export Screaming Frog as UTF-8.
  • No HTML-200 rows — filter to Internal → HTML before export.

Mode A

  • Low high-confidence count — check column mapping and H1 weights.
  • Path/slug dominated — title/H1 columns likely mapped wrong.

Mode B

  • 0 collections — re-export with outlinks/inlinks or use URL patterns.
  • Too many collections — switch from auto-detect to URL patterns.

AI tiebreak

  • 401 — wrong/expired BiFrost key.
  • 404 model — update config/models.json.

  4. Export high-confidence CSV, full review XLSX, or JSON.
